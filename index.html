Flint Resource Manager

C++ May 2025 - May 2025

A small and lightweight resource manager for C++ built using CMake that keeps track of loaded resources and unloads them when appropriate. It allows three storage categories: Manual, Persistent and Volatile.

  • Manual category does not unload the resource until explicitly told to.
  • Persistent category unloads the resource some time after the use count reaches zero.
  • Volatile category unloads the resource when use count reaches zero.

It uses a token object for resource access that increments/decrements the use count for resources on construction/destruction.

https://github.com/yusuf8998/FlintResourceManager

Ibex Engine

C++ Graphics-Programming Game-Dev Jan 2025 - Present

A WIP game engine written in C++ built using CMake. 9th iteration of my custom game engine built for making 3D games with lower graphics quality. My goal is to make a game using minimal libraries and this engine is closer to being complete than my previous iterations. It uses a scene graph system and saves scene nodes using JSON. It currently has built-in nodes for skybox and billboard rendering, light casting and camera. It uses an early version of built-in Ibex Renderer and on hold until the first definitive renderer version. Current version has problems with shadow rendering but otherwise working for general model viewing with Phong lighting model using specular, normal and displacement maps.
